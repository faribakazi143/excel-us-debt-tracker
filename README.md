# US Debt Tracker Dashboard

**Project Overview**  
Analyzed US public and governmental debt data (2015–2023). Built a full interactive Excel dashboard including data cleaning, trend analysis, monthly breakdowns, and future projections.

**Tools & Features Used**
- Power Query for data cleaning
- PivotTables & Pivot Charts
- Slicers and interactive dashboard
- Advanced formulas and conditional formatting
- Forecast function for projections

**Key Highlights**
- Calculated yearly percentage increase in debt
- Identified months with highest and lowest debt increases
- Projected publicly held debt to reach ~$33 trillion by 2027
